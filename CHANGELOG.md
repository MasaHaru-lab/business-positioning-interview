# Changelog

> 这个 changelog 记录这个 skill 从 v1 到 v2 的演变。每次迭代背后都有"骨架之外的隐藏知识"被显性化的故事。
>
> 方法论是被打磨出来的，不是凭空设计出来的。

---

## v2.0.0 — 2026-04-30

> **核心动力**：v1 的"问题列表"是骨架，但**真正决定访谈成败的，是骨架之外的"特殊时刻处理"**。v2 把这些隐藏知识显性化。

### 🆕 新增：阶段 0 加入两个关键采集

- **财务现实四级排序**（A 全家收入 / B 副业 / C 意义 / D 作品）—— 决定整个商业架构走向。v1 是访谈进行 2 小时后才浮现的，太晚。
- **约束盘点**（健康/能量、时间/家庭、平台/地理/语言）—— 用户的物理限制决定产品形态可行域。v1 完全漏问。

### 🆕 新增：8 大特殊时刻处理手册

每个时刻配具体动作清单，不靠访谈者临场反应：

1. **用户情绪化**（突然泪目 / 上头）→ 停下、慢、追问"哪个词"、追问"画面"
2. **用户设定 boundary**（不想展开）→ 完全尊重、归档、找侧面问法
3. **用户用 AI 转引 dodge**（"chat 说我是 X"）→ 看穿不羞辱、解读为低估自己、重新邀请
4. **用户提到外部认证**（行业内人误判为同行）→ 立刻放大、立刻锁定、最终文档独立成洞察
5. **用户给"群体描述"而非"具体的人"** → 强制收紧到一个画得出来的人
6. **用户答得敷衍** → 停下来问"是问题没问到点上吗"
7. **用户中途说"我不知道"** → 提供换角度问法
8. **用户问"你觉得我应该做什么"** → 忍住直接给答案

### 🆕 新增：9 条反模式库

每条配应对话术：

- "我没有长性" → 真实诊断是"架构上瘾型"
- "AI 帮了我所以这不算我的" → AI 时代的协作误判
- "我能看见的都是大牛" → 锚定天花板的不自信陷阱
- "我想先做公益 to start with" → 区分"策略性免费"和"不自信的逃避"
- "普通人对这些不感兴趣" → 用户描述自己亲人的痛点而没意识到
- "我跟别人不一样的是我更用心" → 这是底线不是差异化
- "我的目标客户是 X 群体" → 群不是定位，强制画一个人
- "chat 说我是 X" → 详见特殊时刻处理 #3
- 其他

### 🆕 新增：整合框架问题成为阶段 2 标准工具

> "你看似杂的所有东西其实是一件事——同一个 X 视角的不同应用"

帮助有多兴趣 / 多领域的用户把"杂"重新解读为"整合"。v1 没有这条，导致用户容易陷入"我兴趣太多没有定位"的死胡同。

### 🆕 新增：输出文档的"外部认证证据"独立洞察 + "独有概念库"附录

- 当用户访谈中提到任何一次"行业内人把我误判为同行"，必须独立成一条洞察
- 用户在访谈中创造的专属词汇（如"赛博人"、"丢失了的顿悟"、"以 899 自居"）作为长期 IP 元素入库

### 🔧 修订：响应长度纪律

v1 没有显式约束响应长度，导致访谈者倾向于每轮写大段分析。v2 明确：
- 中段反映：1-3 句 + 1 个问题
- 长综合：只在阶段过渡时出现
- 关键综合（定位句、文档）：可以长，但密度高

### 🔧 修订："画一个人，不画一类人" 升级为硬规则

v1 是软建议，用户经常给"30-50 中年女性"这种群体描述蒙混过去。v2 强制：群体描述不接受，最多两次让步，三次仍是群就回退到上一维度补问。

### 🔧 修订：维度 2 第一题 "你的优点和长处" 加非智识维度 prompt

用户下意识只挑"智识层面"标签（"理科脑"），漏掉 taste / 韧性 / 表达 / 自省 / 操作力等同样重要的维度。v2 强制访谈者给非智识维度的提示词。

### 🔧 修订：维度时长从 60-90 分钟改为 60-150 分钟 + "不要为准时硬压"

实际深度访谈往往超过 90 分钟，效果反而是最好的。强行压缩会破坏"听见自己"的瞬间。v2 明示：内容富时自然延长，用户随时可叫停。

### 🔧 修订：阶段 3 顺序纪律强化

v1 顺序是"用户先尝试 → 我反馈 → 一起改"。v2 明示这是**铁律**——访谈者不要先给综合，让用户先在自己嘴里把第一版定位句搭出来，再用她的原话做骨架。

## v1.0.0 — 2026-04-30

- 输出模板（references/output-template.md）

## 设计原则（贯穿所有版本）

1. **用对方的原话做脚手架**——不替用户"翻译"成商业书面语
2. **看见连接、矛盾、盲区**——单独的问题用户自己也能写问卷，价值在维度之间
3. **温暖但不讨好**——不要"哇好棒"
4. **不灌输理论**——IKIGAI / 定位理论 / 黄金圈是访谈者的内功，不是台词
5. **承认不确定**——v1 文档是初稿，鼓励 7 天后回看 / 3 个月后修订 v2
